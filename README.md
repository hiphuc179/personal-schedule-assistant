# personal-schedule-assistant
Đồ án chuyên ngành: Ứng dụng quản lý lịch trình cá nhân (Python) tích hợp xử lý ngôn ngữ tự nhiên tiếng Việt (NLP) để tự động trích xuất sự kiện.

# Setup instruction

## Setup code

Run the following command step by step

```bash
python -m venv venv

```

Activate venv
```bash
# powershell
.\venv\Scripts\Activate.ps1

# cmd
.\venv\Scripts\activate

# macs/linux
source venv/bin/activate
```

Install dependencies
```bash
pip install -r requirements.txt
```
## Prepare database

copy and rename "data.db.sample" -> "data.db" in database file
